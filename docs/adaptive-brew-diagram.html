🎯 Prediction Pipeline · getPredictionWithContext()
Tier 0 — Bean Model
≥5 bean shots · weighted multivariate regression
grind = avgGrind + doseCoeff×Δdose + ageCoeff×Δage
Fallback doseCoeff=0.05 steps/g when all shots same dose
Bag-change: charShift = currentDelta − lastLearnedDelta
|shift|>0.15 → shift applied, confidence drops to 55%
bean_model · 75% (55% if bag changed)
Tier 0b — Transfer (bean not registered)
Similarity-weighted grind average from similar known beans
similar_beans · 50%
Tier 0.5 — Metadata Prior (0–4 shots)
0–1 shots: globalBaseline + processing×roast table + freshness
2–4 shots: blend(prior, earlyModel) α = 0.4 / 0.6 / 0.8
Dose: (doseIn − refDose) × 0.05 steps/g
Freshness: −0.28 … +0.30 · ageMult ×1.3 for Light roast
bean_prior 35% · bean_early 40/50/60%
Tier 1 — TFLite ML (ENABLE_TFLITE_PREDICTIONS)
Neural net trained on exported shot history (JSON)
ml · 60–95%
Tier 2 — Adaptive KNN
K = √N × density · Gaussian kernel · dist + decay + rating
8-dim: dose · grind · temp · pressure · time · ratio · profile · bean
Post-KNN freshness residual when beanId registered:
Δ = assessFreshness(age) − assessFreshness(DBmeanAge)
knn · 20–85%
Tier 3 — Rules (last resort)
Dose-grind proportionality · freshness residual applied
rules · 20%
Secondary Predictions
KNN k=5 on hypothetical shot → predictedRatio · predictedTime
predictedYield = 18% + (time−28)×0.3 + (ratio−2)×2 · clamp 14–24%
Integer-scale auto-detect: round if all DB grinds are whole numbers
grindAdjustment = predicted − last shot · integer-aware
